Pushbutton LED and OLED Display

A simple embedded system project using STM32F401RE Nucleo board to control an LED and display messages on an OLED screen based on pushbutton inputs.

OLED Display
STM32 Microcontroller
Pushbutton
LED

Objectives

The objective of this project is to create a simple embedded system that demonstrates basic input and output control using the STM32F401RE Nucleo board. The system will read the state of a pushbutton and control an LED accordingly, while also displaying messages on an OLED screen to indicate the button state.

Working Principles

The system operates by continuously monitoring the state of a pushbutton connected to the microcontroller. When the button is pressed, the microcontroller turns on an LED and updates the OLED display to show a message indicating that the button is pressed. When the button is released, the LED turns off and the display updates to show that the button is not pressed. This simple interaction demonstrates how embedded systems can interface with both input devices (the pushbutton) and output devices (the LED and OLED display) to create a responsive user experience.

Results

The system successfully detected button press events and toggled the corresponding LED state while updating the OLED display with the correct messages. The project demonstrated practical GPIO input handling, digital output control and I2C based display interfacing using the STM32F401RE Nucleo board.
