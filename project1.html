Gas Detection & Alarm System

A real-time gas monitoring system using STM32 microcontroller and MQ gas sensor, with visual and audio alarms, LCD display, and Bluetooth communication.

LED Alarm
Buzzer
LCD Display
Bluetooth

Objectives

The objective of this project is to detect the presence of harmful gases using a gas sensor and STM32 microcontroller. When gas is detected above a threshold level, the system triggers alarms including a buzzer and LEDs. Data is sent to a 16x2 LCD display and transmitted wirelessly through an HC-05 Bluetooth module to a mobile device.

Working Principles

The MQ gas sensor outputs an analogue voltage proportional to the gas concentration in the environment. This signal is read by the STM32's ADC. The microcontroller compares the reading against a predefined threshold. If exceeded, the system activates the buzzer and LEDs. The value is displayed on the LCD and transmitted via UART to the HC-05 Bluetooth module for wireless mobile alerts.

Results and Insights

The system successfully detected gas concentrations above the set threshold and triggered both the buzzer and LED alarms in real time. The LCD displayed live sensor readings and the HC-05 Bluetooth module successfully transmitted data to a paired mobile device. The ADC readings were stable and the system responded within milliseconds of gas detection.
