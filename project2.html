EEDE316

OLED Signal Visualizer

An STM32F401RE based signal processing system that displays live graphs of light trends and potentiometer waveform on the OLED display with ADC to DAC feedback and PWN LED control

OLED Display
STM32 Microcontroller
LDR Sensor
PWM LED

Objectives

The objective of this project is to use 0.96 OLED display with the STM32F401RE Nucleo board and visualise real time signal data. The system reads analogue values from an LDR light sensor and a potentiometer processes them through the ADC outputs the results via the DAC and control a PWN LED. All live infromatio including light trends, potentiometer waveforms and the ADC/DAC values are displayed as the graphs on the OLED screen.

Working Principles

The LDR forms a voltage divider with a 10k ohm resistor. As light intensity changes the voltage acroos the LDR changes and is read by the STM32 ADC on one channel. A 10k ohm potentiometer connected to second ADC channel provides a manually adjusable input signal. The ADC readings are fed into the DAC to produce an analogue output, creating a feedback loop for signal processing visualisation. A PWM signal drives an LED whose brighness varies with the potentiometer position. All values are plotted as scrolling graphs on the OLED display using I2C communication.

Components Used

Light sensor forming a voltage divider for ADC input

04

Provides adjustable analogue input for ADC to DAC feedback

05

PWM controlled LED with brightness varying by potentiometer

06

Breadboard and jumper wires

Used for connection of the circuit

Results and Observations

The OLED display successfully provided scrolling graphs showing real time light intensity trends from the LDR and waveform data from the potentiometer. The ADC to DAC feedback loop operated correctly and the PWM LED brightness responded accurately to potentiometer changes. The system demonstrated practical signal processing and visualisation on an embedded system.
